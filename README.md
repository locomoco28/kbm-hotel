 This is a school project of a little hotel management tool by Dogan, Marco and Eren.

## Requirements

- [NodeJS](https://nodejs.org/)

## Installation

```bash
# install dependencies
npm i

# build project
npm run build
```

## Starting Server

```bash
# start server
npm run start
# the server will be listening on port 3000
```
